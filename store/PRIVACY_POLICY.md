# Privacy Policy - Bookmark Manager Pro

**Last Updated:** February 2024

## Overview

Bookmark Manager Pro is a browser extension for managing bookmarks. Protecting your data is our highest priority.

## What Data is Collected?

### Local Data (on your device)
- **Bookmark Tags:** Tags you create or generate are stored locally in your browser (chrome.storage)
- **Notes:** Your personal notes for bookmarks are stored locally
- **Settings:** Your API keys and preferences are stored locally

### External Services (optional)
- **Hugging Face API:** If you use AI tag generation, the bookmark URL and title are sent to the Hugging Face API. This only happens if you have entered your own API key.

## What Data is NOT Collected?

- We do **not** collect any personal data
- We have **no** server of our own
- We use **no** tracking or analytics services
- We do **not** sell any data to third parties

## Permissions

The extension requires the following permissions:

- **bookmarks:** Access to your browser bookmarks for viewing and managing
- **storage:** Local storage of tags, notes, and settings
- **tabs:** To open links
- **contextMenus:** Context menu integration (optional)

## Data Storage

All data is stored exclusively locally in your browser:
- Tags and notes: `chrome.storage.local`
- Settings: `chrome.storage.sync` (can be synced via Chrome Sync)

## Your Rights

You can at any time:
- Delete all stored data via browser settings
- Uninstall the extension (all local data will be deleted)
- Disable AI features by not entering an API key
